# **App Name**: Admission Portal Access

## Core Features:

- Role Selection: User role selection (admin or applicant) via a clear, prominent set of choices.
- NISN Input: NISN input field with clear labels and validation.
- Password Entry: Password input field with secure entry and 'show password' option.
- Authentication: Secure authentication against existing user credentials.
- Persistent Login: Remember me for 2 weeks
- Password Recovery: "Forgot Password" link for password reset flow.

## Style Guidelines:

- Primary color: Sky blue (#72BCD4) to evoke trust and accessibility.
- Background color: Light cyan (#E0F7FA), a very desaturated, bright version of the primary.
- Accent color: Teal (#4DB6AC), for highlighting interactive elements.
- Body and headline font: 'Inter' sans-serif with a modern, machined, objective, neutral look, suitable for both headlines and body text.
- Clean, minimalist layout with ample white space.
- Simple, professional icons for user roles and input fields.
- Subtle transition effects for a smooth user experience.
